# Programming School Landing Page

A modern, responsive landing page for a programming school built with semantic HTML5, optimized for SEO, and designed with accessibility in mind. This static website showcases courses, school information, and provides contact options for prospective students.

## Features

- **Semantic HTML5 Structure**: Built with proper semantic elements (header, nav, main, section, article, footer) for better accessibility and SEO
- **SEO Optimized**: Comprehensive meta tags including Open Graph and Twitter Cards for social media sharing
- **Structured Data**: JSON-LD schema markup for Organization and EducationalOrganization to enhance search engine visibility
- **Accessibility First**: ARIA labels, skip links, and proper heading hierarchy for screen reader compatibility
- **Responsive Design**: Mobile-first approach with viewport configuration for optimal display across all devices
- **Zero Build Process**: Pure HTML/CSS/JavaScript - no compilation or bundling required
- **Fast Loading**: Minimal dependencies and optimized assets for quick page loads
